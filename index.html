Inference¶
Now, let's transform our data suitable for inference. We will do the following:


    

  • Drop the message count for message received. After all, we're only concerned if there's a significant decrease in messages sent after we became a couple.
    • 

  • We will only consider messages sent from April Onwards, as it's already given that Messages sent on March are significantly higher.
    • 

  • Aggregate by date, then map the date to integers from $0 - n$.
    • 



Now, we are interested if there is a significant increase/decrease in messages sent, after my partner said yes. We will assume that the messages that i've sent follows a Poisson Distribution. As we know, this has a parameter $\lambda.$ To solve our problem, We will define it as

$$
\lambda = 
\begin{cases}
\lambda_1  & \text{if } t \lt 56 \cr
\lambda_2 & \text{if } t \ge 56
\end{cases}
$$
if $\lambda_1=\lambda_2$ then it implies that there are no significant change in message volume. How will we get $\lambda_is?$ Remember that $\lambda$ is a parameter from Poisson and hence, it should be a positive real number. We will model this by assuming that it came from a Exponential Distribution with a parameter $\alpha.$


$\alpha$ is a hyper-parameter. We will set the exponential parameter equal to the inverse of the average of the df['Message Count'].


TLDR: We will use PyMC3 to get the probability distribution for 2 lambdas. We will compare their expected values, and if they are equal, then there are no significant increase/decrease in message volume.
